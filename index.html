Welcome to Our Git Page.

In our project we created a todo list application where the user can manage his tasks. the user will log in with a user name and a password and will be able to create, view and delete his tasks.

code explanation

The IToDoListDAO interface lists the methods through which the web application uses the database. we define a class that implement this interface (HibernateToDoListDAO) using Hibernate. The HibernateToDoListDAO class implements the singleton pattern. The database (MySQL) includes the ‘items’ and ‘users’ tables. New users can register and existing users to login/logout. In addition, the simple web application allows the users to manage their own personal todo lists. The creation/deletion/modification were wrote in jsp files and and were designed with the use of bootstrap.
